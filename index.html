Requisitos Funcionais

  • O sistema deve mostras os exemplares que já foram emprestados, e não há uma forma simples de saber quando estarão disponíveis novamente.
  • O sistema deve mostrar a data de devolução e alertar caso esteja perto da data de devolução.
  • O sistema deve mostrar quais livros estamos disponíveis antes do empréstimo.
  • O sistema deve reservar os exemplares antecipadamente.
  • O sistema deve avisar quando a data de devolução estiver próxima.
  • O sistema deve permitir que os alunos acessem e leiam livros digitais por meio de dispositivos móveis, como celulares e tablets.
  • O sistema deve recomendar livros aos usuários com base em seu histórico de leitura.
  • O sistema deve disponibilizar um espaço de discussão para que os usuários possam interagir sobre os livros.
  • O sistema deve permitir que alunos e professores publiquem opiniões sobre os livros e visualizem comentários de outros usuários.

  • O sistema deve disponibilizar uma biblioteca digital integrada ao ambiente escolar.
  • O sistema deve permitir o acesso direto aos livros recomendados dentro da plataforma.
  • O sistema deve registrar os livros acessados por cada aluno e o tempo de leitura, permitindo o acompanhamento do engajamento.
  • O sistema deve permitir a vinculação de atividades complementares aos livros, como questionários, desafios e resumos interativos.
  • O sistema deve identificar e listar os alunos com maior frequência de acesso à plataforma.
  • O sistema deve permitir a participação dos alunos em discursões sobre os livros.

Requisitos Não Funcionais

  • O sistema deve estar disponível para acesso remoto via internet, permitindo seu uso em diferentes locais e dispositivos.
  • O sistema deve fornecer mecanismos de notificação para lembrar os usuários sobre prazos e atividades pendentes.
  • O sistema deve disponibilizar um painel administrativo com informações organizadas e de fácil visualização sobre o uso da biblioteca.
  • O sistema deve garantir que apenas usuários autenticados (alunos e professores) tenham acesso às funcionalidades da plataforma.
  • O sistema deve gerar um rank dos livros mais lidos pelos alunos (se for feito pela escola e não pelo sistema).
  • O sistema deve permitir que os professores indiquem leituras personalizadas para os alunos.
  • O sistema deve disponibilizar que os alunos discutam sobre os livros.
  • O sistema deve garantir acesso de qualquer lugar.

Requisitos Inversos

  • O sistema não deve permitir que livros permaneçam emprestados além do prazo estabelecido sem a aplicação de notificações ou penalidades.
  • O sistema não deve permitir que usuários autorizados utilizem recursos da plataforma de forma indevida ou fora das permissões definidas.
  • O sistema não deve permitir que pessoas externas o acessem (apenas alunos e professores).
  • O sistema não deve permitir desorganização.
  • O sistema não pode ser difícil de acessar
